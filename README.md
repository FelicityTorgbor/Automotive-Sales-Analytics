# Automotive-Sales-Analytics

## Project Overview
This project examines revenue trends, product performance, pricing behavior, order status, and regional sales contributions. The analysis highlights key drivers of revenue and identifies opportunities for market expansion and margin improvement.
---

## Objectives
- Analyze overall sales performance.
- Evaluate geographic revenue distribution.
- Identify top-performing product lines.
- Assess pricing effectiveness against MSRP.
- Examine order and deal-size trends.
- Generate business recommendations.

## Tools Used
- Power BI – Dashboard development and visualization
- Microsoft Excel – Data preparation and cleaning
-DAX (Data Analysis Expressions) – Calculated measures and KPIs
- Power Query – Data transformation and preprocessing
